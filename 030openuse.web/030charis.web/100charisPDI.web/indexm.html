CHARIS Polarization Differential Imaging (PDI)


For detailed information about CHARIS, please refer to the CHARIS website.

Spectro-Polarimetric Imaging is enabled by a Wollaston prism, field stop, and half-waveplate (HWP) rotation for slow modulation. This mode is refered to as CHARIS-PDI (Polarization Differential Imaging).


CHARIS-PDI datacube scanning wavelength. Coronagraph blocks central PSF. Astrogrid calibration speckles enabled.
