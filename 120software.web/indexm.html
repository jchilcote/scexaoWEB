Instrument Control Software

Some of the SCExAO instrument control software is hosted on the SCExAO software repo.

Adaptive Optics Control

SCExAO uses the compute and control for adaptive optics (cacao) package for its AO control.

Data Analysis Software

The CHARIS data reduction pipeline produces calibrated data cubes, and provides some more advanced data reduction capabilities.
