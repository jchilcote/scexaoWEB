Coronagraphs

List of Available Coronagraphs

The SCExAO instrument is equipped with several coronagraphs, which operate across the near-IR (y-K bands) :
  • The Phase induced amplitude apodization complex mask coronagraph (PIAACMC) provides small IWA and high throuput in H band
  • The Vortex coronagraph provides small IWA and high throuput in H band
  • The Lyot coronagraphs provide high throughput, robust performance across the near-IR (y-K bands)
  • The Shaped Pupil coronagraph provides robust performance across the near-IR (y-K bands)
